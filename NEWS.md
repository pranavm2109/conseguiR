# conseguiR NEWS

## 0.99.0

- Refined the package toward an object-first workflow so compute stages return
  in-memory bundles by default and only write outputs when explicitly asked.
- Hardened locus plotting, including dbSNP-backed SNP citation labeling,
  improved label placement, and clearer progress reporting.
- Improved plotting, scoring, and caching behavior across repeated interactive
  sessions.
- Expanded package documentation and refreshed the main vignette/report
  materials.
